## Installation
'''
$ apt-get update && apt-get upgrade
$ apt-get install git texlive-full latexmk
$ git clone https://github.com/Magneto25/CV_Latex.git
$ cd CV_Latex
'''

## Running
'''
$ latexmk -pdf format.tex
'''
